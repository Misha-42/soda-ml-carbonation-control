# 07. Заключение

На текущем этапе НИР была решена задача построения baseline-пайплайна прогнозирования сменных лабораторных показателей процесса карбонизации БСК. На основе 17 Excel-файлов лабораторного контроля за период 20.02.2026–08.03.2026 был сформирован посменный датасет размером 34 строки, где одна строка соответствует одной смене.

Для baseline использовалась единая конфигурация:
- посменный датасет
- RandomForestRegressor
- ручной white-list технологически осмысленных признаков
- временной train/test split

Проведённые эксперименты показали, что лучший baseline получен для target_t:
- MAE = 0.387203
- RMSE = 0.475321
- R2 = 0.036030

Для target_cl также получен положительный результат:
- MAE = 0.733118
- RMSE = 0.806518
- R2 = 0.006915

Наиболее трудным для прогноза оказался target_nh3:
- MAE = 1.334134
- RMSE = 1.535068
- R2 = -0.009601

Таким образом, на текущем этапе:
- pipeline является работоспособным
- baseline для трёх таргетов зафиксирован
- наиболее перспективными для дальнейшего улучшения являются target_t и target_cl
- прогнозирование target_nh3 требует дальнейшего развития признаков, модели и/или расширения данных

Полученные результаты следует рассматривать как baseline-контрольную точку для следующего этапа разработки soft sensor системы.

## Update after linear-vs-RF comparison

На дополнительном этапе было показано, что оптимальная baseline-модель зависит от таргета:
- для target_t лучшей осталась модель Random Forest
- для target_cl лучшей оказалась модель Ridge
- для target_nh3 baseline пока сохраняется на Random Forest

Это усиливает методическую корректность проекта: вместо одной общей модели для всех таргетов зафиксированы лучшие конфигурации отдельно по каждому целевому показателю.
