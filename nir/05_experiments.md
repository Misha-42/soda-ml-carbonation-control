# 05. Эксперименты

## 5.1. Постановка baseline-задачи

На данном этапе решалась задача построения baseline-моделей для трёх сменных таргетов процесса карбонизации БСК:
- target_nh3
- target_cl
- target_t

Исходные данные были собраны из 17 Excel-файлов лабораторного контроля за период 20.02.2026–08.03.2026. Первоначально был сформирован почасовой датасет, однако он был признан методологически слабым для baseline-прогнозирования, поскольку целевые показатели являются сменными, а не часовыми.

В качестве корректной единицы наблюдения была принята одна смена. В результате был собран посменный датасет размером 34 строки (17 дней × 2 смены). Для каждой смены были рассчитаны агрегаты mean/std/min/max по часовым признакам внутри смены.

## 5.2. Выбор baseline-конфигурации

Для baseline использовалась единая конфигурация:
- модель: RandomForestRegressor
- пространство признаков: ручной white-list технологически осмысленных признаков
- схема разбиения:
  - train: даты < 2026-03-01
  - test: даты >= 2026-03-01

Такой подход был выбран как компромисс между методической корректностью, интерпретируемостью и устойчивостью на очень малой выборке.

## 5.3. Результаты baseline по target_nh3

Для target_nh3 были проверены несколько baseline-вариантов, включая:
- почасовой Random Forest
- посменный RF на полном наборе признаков
- RF на top-20 importance
- наивный baseline по среднему
- RF на ручном white-list признаков

Наилучший результат для данного таргета дал вариант rf_shift_whitelist:
- MAE = 1.334134
- RMSE = 1.535068
- R2 = -0.009601

Результат оказался лучше наивного baseline и лучше других проверенных RF-вариантов, однако коэффициент детерминации остался около нуля.

## 5.4. Результаты baseline по target_cl

Для target_cl на том же white-list признаков был получен следующий результат:
- MAE = 0.733118
- RMSE = 0.806518
- R2 = 0.006915

Это означает, что target_cl прогнозируется лучше target_nh3, а модель уже показывает положительное значение R2.

## 5.5. Результаты baseline по target_t

Наилучший результат среди всех трёх таргетов был получен для target_t:
- MAE = 0.387203
- RMSE = 0.475321
- R2 = 0.036030

Таким образом, именно target_t стал лучшим baseline-таргетом на текущем этапе.

## 5.6. Итоговая логика этапа

Проведённая серия baseline-экспериментов показала:
- корректность выбранной сменной постановки задачи
- полезность ручного white-list признаков
- различие в прогнозируемости трёх таргетов

По итогам этапа baseline зафиксирован для всех трёх таргетов, а дальнейшая работа должна быть сосредоточена либо на улучшении target_t и target_cl, либо на расширении данных и признакового пространства.
