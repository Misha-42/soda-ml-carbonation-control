# 08. Ограничения текущего этапа и следующие шаги

## 8.1. Ограничения текущего baseline-этапа

Несмотря на то что baseline для сменных таргетов карбонизации БСК уже построен, зафиксирован и приведён к согласованному виду, текущий этап имеет ряд существенных ограничений.

Во-первых, объём выборки остаётся очень малым. В распоряжении имеется только 34 смены, сформированные на основе 17 дней наблюдений. Это ограничивает статистическую устойчивость выводов и не позволяет рассматривать полученные результаты как окончательные.

Во-вторых, baseline-модели оценивались на одном временном train/test split:
- train: даты < 2026-03-01
- test: даты >= 2026-03-01

Такой подход методически корректен для начального этапа, однако он не заменяет более устойчивую временную валидацию на нескольких последовательных окнах.

В-третьих, качество прогноза по таргетам остаётся умеренным. Даже для лучшего таргета `target_t` значение коэффициента детерминации невелико (`R2 = 0.036030`). Для `target_cl` baseline улучшен после сравнения моделей и наилучший результат получен моделью `Ridge` (`R2 = 0.087208`), однако и этот результат следует рассматривать как baseline, а не как финальное рабочее решение. Для `target_nh3` качество пока остаётся близким к нулевому baseline (`R2 = -0.009601`), что указывает на более высокую сложность данного таргета.

В-четвёртых, текущий набор признаков, хотя и отобран вручную по технологическому смыслу, всё ещё может содержать:
- шумные агрегаты,
- признаки с частичной избыточностью,
- признаки, чувствительные к малому объёму выборки.

В-пятых, на текущем этапе результаты интерпретируются главным образом через статистические метрики (`MAE`, `RMSE`, `R2`) и визуальный анализ ошибок. Для перехода к более прикладной постановке потребуется дополнительно оценить, насколько такие ошибки допустимы с точки зрения технологического процесса и практического использования soft sensor.

## 8.2. Итоговое состояние baseline на текущем этапе

По результатам baseline-этапа зафиксированы лучшие модели по таргетам:
- `target_t` -> `rf_shift_whitelist`
  - MAE = 0.387203
  - RMSE = 0.475321
  - R2 = 0.036030

- `target_cl` -> `ridge`
  - MAE = 0.645223
  - RMSE = 0.773226
  - R2 = 0.087208

- `target_nh3` -> `rf_shift_whitelist`
  - MAE = 1.334134
  - RMSE = 1.535068
  - R2 = -0.009601

Таким образом, на текущем этапе:
- лучший таргет по качеству baseline-прогноза — `target_t`;
- вторым по качеству является `target_cl`;
- наиболее сложным остаётся `target_nh3`.

## 8.3. Следующие шаги исследования

Дальнейшая работа должна быть сосредоточена не на хаотичном расширении числа экспериментов, а на наиболее результативных направлениях.

### 1. Улучшение `target_nh3`
Наиболее логичным следующим шагом является выделение отдельной рабочей ветки для улучшения `target_nh3`, поскольку именно этот таргет остаётся самым слабым. Для него целесообразно:
- пересмотреть white-list признаков;
- проверить более компактные и более устойчивые наборы признаков;
- дополнительно протестировать регуляризованные линейные модели и упрощённые конфигурации;
- отдельно проверить, нет ли проблем с постановкой самого таргета.

### 2. Расширение выборки по сменам
Наиболее сильный резерв качества связан с увеличением числа сменных наблюдений. При добавлении новых данных можно ожидать:
- более устойчивую оценку качества,
- снижение чувствительности моделей к случайным колебаниям,
- более честное сравнение разных алгоритмов.

### 3. Более строгая временная валидация
После накопления дополнительных смен целесообразно перейти к более строгой схеме проверки:
- walk-forward validation,
- несколько последовательных временных окон,
- сравнение среднего качества и его разброса по фолдам.

### 4. Уточнение признакового пространства
Следует отдельно проверить:
- какие агрегаты действительно полезны,
- какие признаки являются слабыми или дублирующими,
- какие признаки дают интерпретируемый технологический сигнал.

### 5. Подготовка к следующему этапу НИР
Так как baseline-пакет уже зафиксирован, следующий этап может идти по двум параллельным направлениям:
- развитие модели и признаков,
- упаковка текущих результатов в итоговый текст НИР и магистерской работы.

## 8.4. Практический вывод

На текущем этапе построен и полностью зафиксирован baseline-пакет для сменных таргетов карбонизации БСК. Полученные результаты нельзя считать финальными, однако они уже подтверждают работоспособность выбранного подхода и наличие полезного предиктивного сигнала по крайней мере для части таргетов. Дальнейшее развитие работы должно быть связано прежде всего с улучшением `target_nh3`, расширением выборки и усилением валидационной схемы.
