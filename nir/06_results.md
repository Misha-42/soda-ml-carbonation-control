# 06. Результаты

## 6.1. Сводка baseline по всем таргетам

На текущем этапе для всех трёх сменных таргетов был построен и зафиксирован единый baseline на основе посменного датасета и модели RandomForestRegressor с ручным white-list признаков.

Итоговые результаты:
- target_t:
  - best_model = rf_shift_whitelist
  - MAE = 0.387203
  - RMSE = 0.475321
  - R2 = 0.036030

- target_cl:
  - best_model = rf_shift_whitelist
  - MAE = 0.733118
  - RMSE = 0.806518
  - R2 = 0.006915

- target_nh3:
  - best_model = rf_shift_whitelist
  - MAE = 1.334134
  - RMSE = 1.535068
  - R2 = -0.009601

## 6.2. Сравнение таргетов

Ранжирование таргетов по качеству baseline-прогноза:
1. target_t
2. target_cl
3. target_nh3

Лучшим таргетом оказался target_t, поскольку для него получены минимальные ошибки и наибольшее положительное значение R2.

Второе место занял target_cl. Несмотря на более высокие ошибки по сравнению с target_t, этот таргет также показал положительное значение коэффициента детерминации, что подтверждает наличие полезного предиктивного сигнала.

Наиболее сложным оказался target_nh3. Хотя white-list baseline улучшил качество относительно других baseline-вариантов для этого таргета, итоговый R2 остался близким к нулю.

## 6.3. Практическая интерпретация

Полученные результаты позволяют сделать следующие выводы:
- выбранный pipeline работоспособен
- white-list технологически осмысленных признаков оправдан
- текущий набор признаков содержит наиболее устойчивый сигнал для target_t и target_cl
- для target_nh3 текущего набора признаков и объёма данных недостаточно для устойчивого прогноза

## 6.4. Ограничения

Следует учитывать, что:
- объём выборки очень мал и составляет только 34 смены
- baseline оценивался на одном временном split
- результаты нельзя считать окончательными
- положительное значение R2 у лучших таргетов пока невелико

## 6.5. Итог этапа

Итогом данного этапа является не финальная оптимальная модель, а корректно построенная baseline-контрольная точка для трёх сменных таргетов карбонизации. Эта точка может использоваться как опорная конфигурация для следующего этапа НИР.
