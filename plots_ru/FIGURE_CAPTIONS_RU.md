## Подписи к рисункам для НИР

### 1. target_t_actual_vs_pred_ru.png
**Рисунок — Сопоставление фактических и прогнозных значений `target_t` на тестовой выборке.**  
На рисунке показано сравнение фактических и предсказанных значений температуры суспензии гидрокарбоната натрия по сменам. Наиболее близкое совпадение факта и прогноза наблюдается именно для `target_t`, что соответствует лучшему качеству baseline-модели среди всех рассмотренных таргетов.

### 2. target_cl_actual_vs_pred_ru.png
**Рисунок — Сопоставление фактических и прогнозных значений `target_cl` на тестовой выборке.**  
График иллюстрирует качество прогноза содержания хлорид-ионов в суспензии гидрокарбоната натрия. Видно, что модель воспроизводит общий характер изменения таргета, хотя точность уступает результату для `target_t`.

### 3. target_nh3_actual_vs_pred_ru.png
**Рисунок — Сопоставление фактических и прогнозных значений `target_nh3` на тестовой выборке.**  
На графике видно, что прогнозирование содержания свободного аммиака является наиболее сложной задачей среди трёх исследованных таргетов. Расхождения между фактическими и предсказанными значениями выражены сильнее, чем для `target_t` и `target_cl`.

### 4. target_t_scatter_actual_vs_pred_ru.png
**Рисунок — Диаграмма рассеяния «факт–прогноз» для `target_t`.**  
Точки, соответствующие предсказаниям модели, располагаются ближе к диагонали `y=x`, что свидетельствует о наилучшем качестве baseline-прогноза для данного таргета.

### 5. target_cl_scatter_actual_vs_pred_ru.png
**Рисунок — Диаграмма рассеяния «факт–прогноз» для `target_cl`.**  
На рисунке показана связь между фактическими и предсказанными значениями содержания хлорид-ионов. Расположение точек вблизи диагонали подтверждает наличие полезного предиктивного сигнала, хотя точность модели остаётся умеренной.

### 6. target_nh3_scatter_actual_vs_pred_ru.png
**Рисунок — Диаграмма рассеяния «факт–прогноз» для `target_nh3`.**  
Для таргета `target_nh3` наблюдается наибольший разброс точек относительно диагонали `y=x`, что подтверждает более низкое качество baseline-прогноза и более высокую сложность задачи.

### 7. target_t_residuals_ru.png
**Рисунок — График остатков модели для `target_t`.**  
По оси абсцисс отложены прогнозные значения, по оси ординат — остатки (`факт − прогноз`). Более компактное распределение точек около нулевой линии указывает на лучшее качество baseline-модели для данного таргета.

### 8. target_cl_residuals_ru.png
**Рисунок — График остатков модели для `target_cl`.**  
Рисунок используется для анализа систематического смещения и структуры ошибок модели. Распределение остатков относительно нулевой линии позволяет оценить устойчивость baseline-прогноза для содержания хлорид-ионов.

### 9. target_nh3_residuals_ru.png
**Рисунок — График остатков модели для `target_nh3`.**  
Для `target_nh3` наблюдается более широкий разброс остатков, что согласуется с худшими метриками качества и указывает на необходимость дальнейшего улучшения признаков и моделей.

### 10. target_t_residuals_hist_ru.png
**Рисунок — Гистограмма распределения остатков для `target_t`.**  
На рисунке представлено распределение ошибок baseline-модели по таргету `target_t`. Более компактное распределение, сосредоточенное около нуля, подтверждает наилучшее качество прогноза для данного показателя.

### 11. target_cl_residuals_hist_ru.png
**Рисунок — Гистограмма распределения остатков для `target_cl`.**  
Гистограмма отражает ширину и возможное смещение распределения ошибок модели при прогнозировании `target_cl`. По сравнению с `target_t` разброс ошибок выше, однако качество остаётся приемлемым для baseline-этапа.

### 12. target_nh3_residuals_hist_ru.png
**Рисунок — Гистограмма распределения остатков для `target_nh3`.**  
На рисунке видно наиболее широкое распределение ошибок, что подтверждает более высокую сложность прогноза `target_nh3` по сравнению с другими таргетами.

### 13. model_compare_t_cl_mae_ru.png
**Рисунок — Сравнение моделей по метрике MAE для `target_t` и `target_cl`.**  
Диаграмма показывает, что для `target_t` лучшей моделью остаётся `Random Forest`, тогда как для `target_cl` меньшая средняя абсолютная ошибка достигается с помощью `Ridge`.

### 14. model_compare_t_cl_r2_ru.png
**Рисунок — Сравнение моделей по метрике R² для `target_t` и `target_cl`.**  
На диаграмме видно, что для `target_cl` регуляризованная линейная модель превосходит `Random Forest`, а для `target_t` лучшим остаётся нелинейный baseline на основе `Random Forest`.
