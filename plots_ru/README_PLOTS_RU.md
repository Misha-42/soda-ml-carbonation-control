# README_PLOTS_RU

В папке собраны русскоязычные версии графиков для НИР.

Основные группы графиков:
- сопоставление факта и прогноза
- диаграммы рассеяния «факт–прогноз»
- графики остатков
- гистограммы остатков
- сравнение моделей по метрикам

Лучшие модели по таргетам:
- target_t -> rf_shift_whitelist
- target_cl -> ridge
- target_nh3 -> rf_shift_whitelist
