## 6.1. Результаты baseline-сравнения моделей

В ходе baseline-эксперимента были получены результаты для четырёх конфигураций моделей машинного обучения. Сравнение показало, что наилучшее качество прогноза продемонстрировала модель `RandomForestRegressor (rf_medium)`.

Полученные значения метрик составили:
- MAE = 0.6245,
- RMSE = 0.9311,
- R² = 0.8904.

Для остальных конфигураций были получены следующие результаты:
- `RandomForestRegressor (rf_small)`: MAE = 0.6369, RMSE = 0.9426, R² = 0.8877;
- `XGBRegressor (xgb_small)`: MAE = 0.7239, RMSE = 1.0579, R² = 0.8585;
- `XGBRegressor (xgb_medium)`: MAE = 0.7475, RMSE = 1.1113, R² = 0.8439.

Таким образом, обе конфигурации `RandomForestRegressor` показали более высокое качество по сравнению с `XGBRegressor` в рамках первого baseline-эксперимента. При этом конфигурация `rf_medium` оказалась лучшей одновременно по всем трём основным метрикам.

Анализ важности признаков для лучшей модели показал, что решающее влияние на результат прогноза оказывает текущий показатель `temp_offgas_k1`. Вклад остальных признаков оказался значительно ниже. Это свидетельствует о выраженной инерционности процесса на коротком горизонте прогноза: значение температуры отходящего газа через 6 минут в значительной степени определяется её текущим значением.

Полученный результат подтверждает, что baseline-модель в первую очередь использует автокорреляционные свойства целевого сигнала. С одной стороны, это позволяет получить достаточно высокое качество прогноза уже на первом этапе исследования. С другой стороны, такой результат указывает на необходимость дальнейших экспериментов, направленных на выявление вклада остальных технологических параметров.

В качестве следующего этапа целесообразно:
- сравнить качество прогноза при наличии и отсутствии признака `temp_offgas_k1` во входных данных;
- расширить признаковое пространство за счёт лаговых признаков и скользящих характеристик;
- протестировать альтернативные целевые переменные, в том числе `temp_slurry_k1` на следующий шаг времени.

## 6.2. Анализ влияния доминирующего признака `temp_offgas_k1`

Для оценки вклада текущего значения температуры отходящего газа в точность прогноза был проведён дополнительный эксперимент. Рассматривались два варианта модели:
1. с включением признака `temp_offgas_k1` во входное пространство;
2. с исключением признака `temp_offgas_k1` из набора признаков.

В варианте с использованием `temp_offgas_k1` наилучшее качество показала модель `RandomForestRegressor (rf_medium)`:
- MAE = 0.6245,
- RMSE = 0.9311,
- R² = 0.8904.

После исключения признака `temp_offgas_k1` наилучшей стала модель `XGBRegressor (xgb_small)`, однако качество прогноза существенно снизилось:
- MAE = 1.5904,
- RMSE = 2.2084,
- R² = 0.3833.

Таким образом, исключение признака `temp_offgas_k1` привело к резкому ухудшению всех основных метрик качества. Это подтверждает, что на рассматриваемом горизонте прогноза 6 минут текущее значение температуры отходящего газа является основным источником информации для модели.

Полученный результат указывает на выраженную инерционность процесса: ближайшее будущее значение целевого сигнала в значительной степени определяется его текущим состоянием. Остальные технологические параметры сохраняют определённую прогностическую ценность, однако их вклад в рамках baseline-подхода существенно ниже.

Дополнительно установлено, что при наличии доминирующего признака лучшие результаты показывает `RandomForestRegressor`, тогда как после его исключения более устойчивой оказалась модель `XGBRegressor`. Это может свидетельствовать о различии в способности моделей извлекать слабые нелинейные зависимости из оставшегося признакового пространства.
