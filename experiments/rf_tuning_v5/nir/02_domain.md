# 02. Предметная область

## 1) Что сюда писать
- Кратко описать технологический процесс производства соды.
- Перечислить ключевые параметры процесса, которые важны для анализа.
- Указать, какой показатель прогнозируется (target).

## 2) Какие таблицы и графики сюда вставлять
- Таблица «параметр процесса -> смысл -> единицы измерения».
- Простой график изменения target во времени (если есть временная колонка).

## 3) Какие данные из проекта будут использоваться
- Колонки исходного CSV из `data/`.
- Информация о целевой колонке и временной колонке из запуска `train_baseline.py`.
