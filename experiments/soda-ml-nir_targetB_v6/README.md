# Baseline-контур для прогнозирования `target1`

## Краткое описание

Проект содержит результаты серии экспериментов по построению воспроизводимого baseline-контура машинного обучения для технологического показателя `target1`. Тестировались конфигурации моделей, признаковых пространств, отбор признаков и устойчивость.

## Ключевые результаты

**Итоговый baseline:**
- **Модель:** `RandomForestRegressor` (базовые параметры).
- **Признаковое пространство:** `w60_only`.
- **Отбор признаков:** `top_100` по `feature importance` на train.
- **Улучшение от Exp.2:** RMSE с 4,116668 до 3,996800; R² с 0,113823 до 0,164679; признаки с 769 до 100.

**Устойчивость:** `TimeSeriesSplit(n_splits=4)`.

## Ход экспериментов

### 6.1. Сравнение моделей (1538 признаков)

`Ridge` не подходит (отрицательный R²). Лучше `Random Forest`: MAE=3,347818, RMSE=4,150467, R²=0,099212 [web:31].

### 6.2. `w60_only` (769 признаков)

MAE=3,365045, RMSE=4,116668, R²=0,113823. Улучшение за счет снижения избыточности.

### 6.3. Light tuning RF

MAE=3,397348, RMSE=4,217209, R²=0,070008. Тюнинг без отбора признаков ухудшил.

### 6.4. `mean + last only` (256 признаков)

MAE=3,245783, RMSE=4,156987, R²=0,096379. Компактный резерв.
