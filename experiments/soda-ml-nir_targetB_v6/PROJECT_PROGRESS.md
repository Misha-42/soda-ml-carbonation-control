- delta R² = +0.050856
- число признаков снижено: 769 → 100

- mean RMSE = 4.157308
- std RMSE = 0.922905
- mean R² = 0.146179
- std R² = 0.076943

Вывод:
- baseline для `target1` выбран
- `exp_05 top_100` является новым рабочим и рабоче устойчивым baseline
- дополнительная серия top-N экспериментов не требуется

Walk-forward проверка baseline завершена.

Проверяемая конфигурация:
exp_05 = w60_only + Random Forest + фиксированный набор top_100 признаков,
ранее отобранных по feature importance на train.

Результаты по временным фолдам:
- fold_1: RMSE = 2.781229, R² = 0.019812
- fold_2: RMSE = 5.172466, R² = 0.216860
- fold_3: RMSE = 4.755381, R² = 0.166539
- fold_4: RMSE = 3.920155, R² = 0.181504

Итог:
- на всех фолдах R² остался положительным
- системного развала качества не наблюдается
- разброс RMSE присутствует, что ожидаемо при малом объёме данных и небольших train-окнах на ранних фолдах

Вывод:
exp_05 top_100 принимается как новый рабочий и условно устойчивый baseline
для дальнейшего переноса на следующий контур.

Методическое уточнение:
в walk-forward проверялась устойчивость модели на фиксированном заранее выбранном наборе top_100 признаков, а не полный pipeline с переотбором признаков внутри каждого фолда.
